## 二、已完工能力

### 核心架构
- ✅ **PlatformAdapter 插件架构** — `src/platforms/base.py` 280行，抽象基类 + `PLATFORM_REGISTRY`，三步入驻新平台
- ✅ **DouyinAdapter** — URL解析（短链→302→ID）/ 单视频（无水印）/ 翻页 / 喜欢列表 / 评论 / 收藏夹 / 合集（mix_info 替代方案）全实现
- ✅ **异常体系** — `src/exceptions.py` 9种异常类（对照 Evil0ctal 7种已覆盖）
- ✅ **Sign-server** — Node.js 独立签名服务，`bootstrap.js` + `server.js` + 诊断工具链，端口自动探测
- ✅ HTTP 直连反爬 — Cookie + 设备指纹，不需要 a_bogus / msToken

### 桌面体验
- ✅ PyQt6 原生 GUI — `main_window.py` 27KB，含 pages/dialogs/widgets/自定义字体
- ✅ 暗色/亮色主题切换（Fusion style）
- ✅ 系统托盘 + 快捷键（`Ctrl+H` 首页 / `Ctrl+,` 设置 / `Ctrl+Q` 退出 / `Esc` 最小化）
- ✅ 单实例检测（QLocalSocket）
- ✅ WebView 扫码登录（QtWebEngine）
- ✅ 中文翻译（`translations/`）

### 工程化
- ✅ Inno Setup 安装器 — 含卸载保留数据选项、exe 名去版本号
- ❌ SignPath 代码签名（未成功）
- ✅ 自动更新（`version.json` → GitHub Release）
- ✅ 钉钉反馈系统 — 自动附带诊断日志
- ✅ Cookie 管理 + `settings.json` 持久化

### 社区展示
- ✅ README 全面美化 — typing SVG / Badge 矩阵（12枚）/ 演示 GIF / 访问计数器 / Star History / 贡献者墙 / Activity Graph / 贪吃蛇提交图
- ✅ 项目 Logo（`logo.png` + `app.ico`）
- ✅ `注意事项.md` / `同类项目对比分析.md` / `Origami美化方案.md` — 三份内部参考文档
