## v0.6.2 改动

### 体验优化
- 浏览器动态发现: `where` → 注册表 → 常见路径，Chrome/Edge 全支持
- 反馈附诊断日志: 自动带 `_sign_err.log` / `_sign_debug.log` / `_instance.log` 尾部 5000 字符
- 去掉登录成功弹窗: 扫码完右上角直接变头像
- UI 不再卡顿: 后台线程日志走 `pyqtSignal`，`start_server` 只等 0.3s
- 下载路径即时生效: 设置页改路径不用重启
- 路径加重置按钮: 一键恢复默认
- 卸载弹窗: 询问保留/删除 `settings.json` + `output/`
- 静默 libpng iCCP 警告

### 稳定性
- 冷启动 Chrome 不崩: 最小 PATH + Node 端动态加浏览器目录
- node 僵尸进程: `stop_server` + `was_running` 优化 + 启动扫孤儿
- 退出不崩溃: Python 标志替代 C++ widget 访问
- `os._exit` 前清理 sign-server: 4 层防护

### Bug 修复
- 批量下载水印: 下载前调 sign-server 刷新详情
- 画廊取消后再选死锁: `selected_indices` 已设时跳过弹窗
- 单作品图集进度条卡 66%: `total` 排除幻灯片视频
- 批量目录显示 0 作品: `rglob` 找 `.downloaded.json`
- 预览弹窗卡顿: 分批加载 (12 个/批, 15ms 间隔)
- 端口 9876 被封: 自动探可用端口
- 设置页路径不生效: 补 `load_settings` 导入
