# 功能脑暴

> 相关：[[tests/README|测试总结]] · [[../README|项目主页]] · [[../同类项目对比分析|竞品分析]] · [[../注意事项|合规约束]]

## 已有能力

- 单视频/图集/实况下载（无水印）
- 批量主页作品下载（视频/图集/实况自动分类）
- 用户主页完整信息（粉丝/IP属地/标签/封面/电商等 40+ 字段）
- 喜欢列表翻页（自己的）
- 评论区内容 + 表情包图片下载
- 文章/长文正文提取
- 合集/短剧归属检测（via mix_info）
- 剪贴板自动识别 + 窗口置顶

---

## 用户画像 + 功能建议

### 📦 内容收藏家

> "刷到喜欢的视频/图集想保存下来，怕哪天被删了"

| 功能 | 说明 |
|------|------|
| 一键保存全部喜欢 | 翻页拉取喜欢的作品，批量下载 |
| 增量同步 | 只下载新增的喜欢，已有跳过 |
| 分类归档 | 按作者/标签自动分文件夹 |
| 收藏夹导出 | 把自己创建的收藏夹整夹导出 |

### 📊 数据分析者

> "想研究一个账号的数据表现，做竞品分析"

| 功能 | 说明 |
|------|------|
| 作品数据报表 | 导出 CSV：发布时间/点赞/评论/分享/收藏，做趋势图 |
| 粉丝画像 | 从评论中提取用户地域、性别分布 |
| 爆款识别 | 标记点赞/评论比异常高的作品 |
| 热评分析 | 按点赞排序导出所有评论 |
| 对比报告 | 两个账号的数据横向对比 |

### 💬 评论研究者

> "评论区才是精华，想保存有趣的评论和表情包"

| 功能 | 说明 |
|------|------|
| 评论全量导出 | Markdown/CSV 格式，含楼层、回复 |
| 表情包收割机 | 自动下载评论区所有贴纸/表情包 |
| 评论词云 | 提取高频词生成词云图 |
| 热门评论TOP榜 | 按点赞数排序输出 |

### 📖 内容整理者

> "想把抖音当知识库，系统性整理内容"

| 功能 | 说明 |
|------|------|
| 文章下载 | 保存长文为 Markdown（标题+正文+头图） |
| 作品目录索引 | 下载后自动生成带缩略图的 HTML 目录页 |
| 合集打包 | 检测作者的合集/短剧，整系列下载 |
| 自动标签 | 从作品描述提取 #标签，生成分类索引 |
| 全文搜索 | 对下载的描述/文章建立本地搜索 |

### 🔄 自动化场景

> "不想手动操作，后台静默干活"

| 功能 | 说明 |
|------|------|
| 关注列表监控 | 检测关注的人发了新作品，自动下载 |
| 定时任务 | 每天/每周自动同步指定账号 |
| RSS 输出 | 把下载的作品生成 RSS feed |
| 系统托盘后台 | 最小化到托盘，剪贴板检测静默下载 |
| 多账号管理 | 切换不同 Cookie 对应不同抖音号 |

### 🎨 进阶玩法

| 功能 | 说明 |
|------|------|
| 无水印直链分享 | 复制下载链接直接发给别人 |
| 视频截图墙 | 批量生成视频缩略图时间轴 |
| 音频提取 | 只下载视频的 BGM/音频 |
| 去重 | 同作品跨作者检测去重 |
| 格式转换 | 下载后自动转 webp→jpg、heic→png |

## 优先级建议

1. 🔴 **高** — 喜欢列表增量同步、作品数据 CSV 导出
2. 🟡 **中** — 评论导出、文章保存、合集打包
3. 🟢 **低** — 关注监控、RSS、多账号
